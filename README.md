# 💬 priconnecte

Полнофункциональный веб-мессенджер с современным дизайном и расширенными возможностями. Написан на чистом HTML, CSS и JavaScript.

## ✨ Все функции

### 📱 Основные
- ✅ Отправка текстовых сообщений
- ✅ Голосовые сообщения (запись и отправка)
- ✅ История переписки (LocalStorage)
- ✅ Индикатор набора текста
- ✅ Временные метки

### 👤 Профиль
- ✅ Изменение аватара (загрузка файла)
- ✅ Изменение никнейма
- ✅ Персональный статус

### 💬 Чаты
- ✅ Создание новых чатов
- ✅ Добавление контактов
- ✅ Закреплённые чаты
- ✅ Архивация чатов
- ✅ Уведомления о непрочитанных

### 📝 Сообщения
- ✅ Удаление сообщений
- ✅ Редактирование сообщений
- ✅ Пересылка сообщений
- ✅ Избранные сообщения
- ✅ Ответ на сообщение
- ✅ Предпросмотр ссылок

### 🔍 Поиск
- ✅ Поиск по контактам
- ✅ Поиск по сообщениям в чате
- ✅ Навигация по результатам

### 🎨 Интерфейс
- ✅ 3 темы (Тёмная/Светлая/Фиолетовая)
- ✅ Эмодзи-пикер
- ✅ Адаптивный дизайн
- ✅ Контекстные меню
- ✅ Анимации и переходы

### ⚙️ Настройки
- ✅ Выбор темы
- ✅ Уведомления браузера
- ✅ Ночной режим (авто)
- ✅ Звуковые уведомления

### 📤 Экспорт
- ✅ Экспорт в TXT
- ✅ Экспорт в JSON
- ✅ Экспорт в HTML

## 🚀 Запуск

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
3. Готово!

## 🛠 Технологии

- HTML5
- CSS3 (Variables, Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- LocalStorage API
- MediaRecorder API (голосовые)
- Notification API
